# Ghostty themes

Custom color themes for Ghostty terminal.

## Themes

### cosmic

Space-inspired theme with deep violet, dusty rose, and cool aqua-green colors against a near-black background.

### synthwave

Retro 80s aesthetic with hot pink, electric teal, and neon violet on a deep synth-blue background.

### abyssal

Deep ocean theme featuring bioluminescent accents with cool blues and glowing aqua tones.

### volcanic

Molten lava and dark earth tones with fiery oranges and deep amethyst against volcanic rock.

### neon

Vibrant cyberpunk theme with electric pink, glowing lime green, and dazzling neon yellow.

## Usage

Add to your Ghostty config:

```shell
theme = <theme-name>
```

Example:

```shell
theme = cosmic
```
